# COVID-19 Studies

This repository contains R and Python code used in COVID-19 epidemiologic and clinical research projects. 
It includes scripts for data preprocessing, cohort construction, 1:1 matching, Cox proportional hazards modeling, and generation of descriptive summary tables (e.g., Table 1). 

---

## Requirements

### R
- R (≥ 4.0)
- Recommended packages:
  - `MatchIt`
  - `survival`
  - `tableone`
  - `tidyverse`
  - `lubridate`

### Python
- Python (≥ 3.8)
- Recommended packages:
  - `pandas`
  - `numpy`
  - `lifelines`
  - `matplotlib`
  - `scikit-learn`


## Data Availability

Data used in these analyses are **not publicly available** due to institutional policies governing patient data sharing.  
Datasets may be made available from the corresponding author **upon reasonable request**, pending institutional approval and execution of a data use agreement.
